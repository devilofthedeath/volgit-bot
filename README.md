## FAQ
Ответы пустые, это было сделано намеренно в целях недопущения неточностей.
Для редактирования FAQ:
1. открыть файл data/faq_data.py
2. в поле **answer** в нужной категории вписать правильный ответ на нужный вопрос

## Навигация
Могут быть некоторые неточности, связанные с навигацией
Для редактирования координат / названий точек:
1. открыть файл data/navigation_data.py
2. в LOCATIONS в полях name и coords вписать нужные значения

# Дополнительно
## Текущая архитектура и потенциал для масштабирования

Проект реализован с модульной архитектурой, готовой к расширению:

### База данных (заготовка)
- **`bd_handler/`** - базовый слой для работы с БД
  - `__init__.py` - инициализация подключения
  - `bf_class.py` - основные ORM-модели и запросы

### API-интеграции (заготовка)  
- **`api_integrations/`** - модуль для внешних API
  - Интеграция с сайтом олимпиады
  - Загрузка данных в реальном времени
  - Синхронизация расписания

### Система прав доступа
- **`filters/`** - фильтры и middleware
  - `__init__.py` - регистрация фильтров
  - `is_admin.py` - система админ-прав
